The Brand Builder app has been pivoted into a Professional Brand Photography Gallery.
Instead of fixed advertising categories, the app now focuses on generating a diverse set of high-end product shots that showcase your brand from multiple professional perspectives.
New Features:
Diverse Photography Gallery: Every generation now produces 4 distinct professional shots:
Studio Hero Shot: A clean, minimalist commercial shot with perfect lighting.
Lifestyle Context: The product placed in a modern, high-end environment (e.g., marble or wood surfaces).
Macro Detail: A close-up shot focusing on textures, materials, and craftsmanship.
Dynamic Angle: A cinematic, low-angle shot for a bold and powerful brand presence.
Multimodal Input: You can still upload a reference photo or simply describe your product to get started.
Professional Consistency: All shots are generated in parallel to provide a cohesive visual set for your brand.
Brand Name Overlay: You can still insert your brand name to be added as a professional overlay when you download any of the images.
Strict "No-Human" Policy: Every prompt is engineered to ensure the focus remains 100% on the product, with absolutely no people in the images.

I've expanded the Brand Builder by Shanto gallery with two additional professional photography styles, bringing the total to 6 distinct shots. These new views are specifically designed to enhance your product's beauty using relevant artistic props and sophisticated compositions.
New Photography Styles:
Atmospheric Mood: A cinematic shot using soft fog, dramatic shadows, and elegant lighting. It incorporates artistic props like silk fabric or light prisms to create a dreamy, high-end feel.
Structured Composition: A balanced, architectural shot where the product is arranged with geometric props like stone plinths, spheres, and minimalist blocks for a sophisticated look.
Complete Gallery Overview:
Studio Hero Shot: Clean, minimalist commercial focus.
Lifestyle Context: Modern environment (marble/wood) for real-world luxury.
Macro Detail: Extreme close-up on textures and craftsmanship.
Dynamic Angle: Bold, low-angle cinematic composition.
Atmospheric Mood (New): Soft lighting and elegant props.
Structured Composition (New): Geometric balance and architectural props.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
